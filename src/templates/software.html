Software - Sistema Operativo (SO)

Autore: Antonio Pierro / @an_pierro

Indice

  • Definizione del sistema operativo (SO);
  • Gestione attività;
  • Prompt dei comandi.

Definizione del SO

  • Un sistema operativo è un programma che agisce come intermediario tra utente e hardware;
  • In generale non svolge un compito evidente all'utente, ma fornisce funzionalità necessarie alle applicazion;
  • Esempio: stampa un archivio ("file").

Sistemi operativi

  • Esistono vari tipi di sistemi operativi;
  • I principali s.o. per personal computer sono:
    • Windows di Microsoft;
    • Mac OsX di Apple;
    • Linux, open source.

Com'è fatto un sistema operativo

  • È fatto di software, di solito scritto in linguaggio C;
  • È molto complesso. Windows XP contiene 40 milioni di linee di codice;
  • Risiede di solito sul disco rigido, in directory specifiche.

Funzioni del sistema operativo 1/2

  • Facilitazione: semplifica l'uso del sistema nascondendo i dettagli di macchina;
  • Esempio: la metafora desktop, che fa somigliare lo spazio di lavoro ad una scrivania.

Funzioni del sistema operativo 2/2

  • Parallelismo: consente di far coesistere più utenti e programmi, che utilizzano a turno risorse e dispositivi;
  • Protezione: fornisce strumenti di sicurezza che proteggono le risorse;
  • Esempio: il SO impedisce che un solo programma usi il processore, o che un utente possa spiare cosa fa un altro utente, o che un programma distrugga i dati di un altro.

Storia dei sistemi operativi

  • (1945-1955) I primi computer erano fatti di valvole e risolvevano problemi di tipo militare;
  • Ogni poche ore si rompeva una valvola;
  • Niente sistema operativo, nè linguaggi;
  • I programmi erano inseriti da console usando interruttori per accendere bit in memoria.

Task Manager

  • Il sistema operativo windows mette a disposizione un programma (Task Manager o Gestione Attività) che consente di:
    • visualizzare informazioni relative a programmi e processi attivi;
    • monitorarne le prestazioni in termini di uso di memoria e tramite altri indicatori;
    • accedere a elaborazioni statistiche sulle performance della memoria e della CPU.
  • Per avviare il programma occorre:
    • premere contemporaneamente i tasti ctrl, alt, canc (o del);
    • nella schermata che comparirà, scegliere la voce "avvia Gestione Attività".

Analisi delle performance

  • La scheda "Prestazioni" del task manager mostra informazioni statistiche, sull'utilizzo della CPU e della mamoria;
  • I due grafici in alto sono relativi alla CPU, i due in bassi sono invece relativi alla RAM.

Prompt dei comandi 1/2

  • Il prompt dei comandi, o shell, è una funzionalità di Windows che consente agli utenti di eseguire comandi, tra cui quelli MS‐DOS (Microsoft Disk Operating System)

Directory di lavoro

  • È possibile modificare la cartella di lavoro eseguendo il comando cd seguito da:
    • un nuovo percorso assoluto, ad esempio digitando: C:\>cd D:\User\Anna;
    • un percorso relativo alla cartella corrente, ad esempio il comando: C:\>cd Desktop;
    • due punti singoli, per tornare indietro di un livello: C:\> cd ..

Comandi per la gestione dei files

dir visualizza l'elenco dei file e delle directory di una cartella;
mkdir nome_cartella (make directory) crea una cartella a cui viene attribuito il nome specificato;
rmdir nome_cartella rimuove la cartella specificata, ma soltanto se è vuota (ovvero non contiene alcun file nè sottocartelle);
del nome_file rimuove il file specificato;
del nome_cartella: rimuove la cartella specificata e tutti i files che contiene;
tree visualizza attraverso una struttura ad albero la gerarchia di sottocartelle della cartella corrente
ECHO C:> ECHO ciao >hello.txt oppure C:> ECHO come stai? >> hello.txt
TYPE TYPE hello.txt visualizza il contenuto del file
HELP HELP DIR: per ottenere informazioni specifiche sull'utilizzo del comando di interesse.

Tipi di documenti

I dati

  • I dati, siano essi documenti o altro, sono memorizzati in archivi (file);
  • Un file è una sequenza di byte e può rappresentare qualsiasi insieme di dati (numerici, alfabetici, audio, video) in base ad uno o più codici;
  • i file di testo si basano sul codice ASCII.

Attributi dei file

Nome Nome simbolico con cui l'utente può riconoscere il file.
Tipo Stabilisce con quale programma potrebbe essere aperto il programma.
Posizione Indica la poszione all'interno del file di sistema.
Dimensione Numero di byte occupati dal file.
Protezione Diritti di accesso al file per gli utenti del sistema.
Data Data di creazione e/o di modifica.

La struttura del filesystem

  • L'insieme delle informazioni memorizzate sulla memoria di massa è organizzato secondo una struttura che va sotto il nome di filesystem;
  • I sistemi operativi utilizzano i concetti comuni di file e directory;
  • Un file è un insieme di informazioni che possono rappresentare dei dati (un testo, un'immagine, una canzone, un film, ...);
  • Una directory è una sorta di contenitore che può contenere sia file che altre directory.

Struttura gerarchica ad albero

  • Poiché le directory possono contenere sia file che altre directory, si viene a delineare una struttura gerarchica ad albero dell'organizzazione dei file e delle directory, detto directory tree o albero delle directory;
  • Un filesystem è formato quindi dall’insieme dei file e delle directory e dalla loro organizzazione.
