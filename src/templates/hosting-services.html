Web hosting

  • In informatica si definisce hosting (dall'inglese to host, ospitare) un servizio di rete che permette di allocare su un server web le pagine web di un sito web.
  • Le pagine web inserite saranno così accessibile dalla rete Internet e ai suoi utenti.
  • Tale "server web", definito "host", è connesso ad Internet in modalità idonea a garantire l'accesso alle pagine del sito mediante il web browser.
  • Attualmente i browser più usati sono: Google Chrome, Internet Explorer e Mozilla Firefox

GitHub

  • GitHub è un servizio web di hosting che può essere utilizzato per lo sviluppo di un sito web per promuovere la nostra azienda.
  • Il sito fornisce funzionalità simili a un social network come feeds e follower.

Utilizzo di GitHub

  • Registrazione;
  • Creazione del primo sito;
  • Modifica delle pagine web;
  • Accesso a GitHub;
  • Aggiunta di immagini.

Bitbucket

  • Come GitHub, Bitbucket è un servizio web di hosting che può essere utilizzato per lo sviluppo di un sito web per promuovere la nostra azienda;
  • A differenza di GitHub le pagine vengono pubblicate istantaneamente appena create;
  • Secondo gli utilizzatori meno esperti l'interfaccia grafica di Bitbucket risulta più facile da utilizzare rispetto a quella di GitHub.

Segnalare un sito ai motori di ricerca

  • I motori di ricerca sono un mezzo importantissimo per chi gestisce un sito Internet.
  • Avere un sito ben posizionato su Google, Bing, ecc. significa comparire nei primi posti dei risultati delle ricerche ed aumentare sensibilmente le probabilità che nuovi utenti arrivino sulle nostre pagine cercando un determinato prodotto.

Segnalare un sito a Google

  • Per segnalare un sito a Google digitare il seguente indirizzo: https://www.google.com/webmasters/tools/submit-url e inserire le informazioni relative al nostro sito:
  • Dopo aver inserito le informazioni premere il tasto "Submit request".

Segnalare un sito a Bing

  • Per segnalare un sito a Bing digitare il seguente indirizzo: http://www.bing.com/toolbox/submit-site-url e inserire le informazioni relative al nostro sito:
  • Dopo aver inserito le informazioni premere il tasto "Submit".

Google Map Maker

  • Google Map Maker è un servizio collaborativo progettato e distribuito da Google
  • Gli utenti possono contribuire al miglioramento delle mappe, inserendo nuovi dati geografici o modificando quelli esistenti su Google Map
  • Le modifiche, prima di essere visibili a tutti, devono essere rivedute da altri utenti e convalidate dalla stessa Google, onde evitare vandalismi, doppioni ed errori.

Accesso a Google Map Maker

  • Digitare sulla barra degli indirizzi http://www.google.com/mapmaker
  • Per accede a Google Map Maker bisogna avere un account su Google
  • Premere sul punsante rosso "Sign In"

Aggiunta di un luogo

  • Dopo essersi autenticati è possibile aggiungere la nostra azienda su "Google Map" premendo il tasto "Add Place"
  • Sposta il cursore sul luogo desiderato, poi fai clic una volta per rilasciare il segnaposto.

Google Analytics

  • Google Analytics è un servizio Web gratuito di Google che può essere usato per analizzare le statistiche sui visitatori del sito web della nostra azienda.
  • Sul sito è possibile monitorare i visitatori che accedono al nostro sito (sesso, età, paese di provenienza, ecc.)

Accesso a Google Analytics

  • Scrivere nella barra degli indirizzi www.google.com/analytics/ come mostrato in figura:
  • Premere il tasto "Access Google Analytics"

Implementazione

  • Google Analytics viene implementato aggiungendo un frammento di codice JavaScript che l'utente aggiunge in tutte le pagine del proprio dominio web.
