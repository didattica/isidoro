Web hosting

  • In informatica si definisce hosting (dall'inglese to host, ospitare) un servizio di rete che permette di allocare su un server web le pagine web di un sito web.
  • Le pagine web inserite saranno così accessibile dalla rete Internet e ai suoi utenti.
  • Tale "server web", definito "host", è connesso ad Internet in modalità idonea a garantire l'accesso alle pagine del sito mediante il web browser.
  • Attualmente i browser più usati sono: Google Chrome, Internet Explorer e Mozilla Firefox

GitHub

  • GitHub è un servizio web di hosting che può essere utilizzato per lo sviluppo di un sito web per promuovere la nostra azienda.
  • Il sito fornisce funzionalità simili a un social network come feeds e follower.

Utilizzo di GitHub

  • Registrazione;
  • Creazione del primo sito;
  • Modifica delle pagine web.

Registrazione a GitHub

  • Collegarsi al sito https://github.com
  • Inserire nei tre campi vuoti i senguenti valori: nome dell'azienda, email e password come mostrato nella figura.
  • Premere il tasto verde.

Scelta del piano tariffario

  • Scegliere il piano che prevede zero costi (free) e premere il tasto verde.

Scelta dell'indirizzo web dell'azienda

  • Premere sul tasto "+" e poi il tasto "New repository" come mostrato in figura:
  • Nella schermata successiva scrivere il nome a cui corrisponderà l'indirizzo della nostra azienda (l'estensione .github.io è obbligatoria). Per esempio:
  • Dopo aver scritto il nome, premere il tasto verde.

Creazione del sito attraverso modelli esistenti

  • Nella schermata a destra premere sul pulsante "Settings"
  • Nella schermata successiva premere sul pulsante "Automatic Page Generator"

Modifica del nome dalla pagina

  • Nella schermata successiva è possibile modificare il nome della pagina web semplicemente editando in seguente campo
  • Dopo aver modificato il titolo della pagina, pigiare sul tasto verde (Continue to layouts)

Selezionare il modello

  • Selezionare il modello che si preferisce premendo sulle immagini in miniatura come mostrato nella figura:
  • Premere sul tasto verde "Publish page"

Accedere al sito web

  • Il sito della nostra azienda in pochi minuti sarà accessibile da internet
  • L'indirizzo dipenderà dal nome con cui ci siamo registrati si GitHub
  • Nell'esempio mostrato l'indirizzo sarà:
    http://agriturismo-sardegna.github.io

Bitbucket

  • Come GitHub, Bitbucket è un servizio web di hosting che può essere utilizzato per lo sviluppo di un sito web per promuovere la nostra azienda;
  • A differenza di GitHub le pagine vengono pubblicate istantaneamente appena create;
  • Secondo gli utilizzatori meno esperti l'interfaccia grafica di Bitbucket risulta più facile da utilizzare rispetto a quella di GitHub.

Registrazione a Bitbucket 1/2

  • Collegarsi al sito https://bitbucket.org
  • Inserire nei tre campi vuoti i senguenti valori: nome dell'azienda, email e password come mostrato nella figura.
  • Premere il tasto blu.
